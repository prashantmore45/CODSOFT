# Online Quiz Maker

> **Important (2026-04-14):** This folder is kept as an **internship snapshot** only.  
> The actively maintained project has moved to: https://github.com/prashantmore45/online-quiz-maker

This project is developed as part of the **CodSoft Web Development Internship (Level 2 Task)**.

## 🚀 Features
- User Registration & Login (JWT Authentication)
- Create Quizzes with Questions & Multiple Options
- Browse Available Quizzes
- Attempt Quiz (One Question at a Time)
- Instant Result Calculation
- Protected Routes
- Mobile Responsive UI

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

## ▶️ How to Run the Project Locally

### Backend
```bash
- cd backend
- npm install
- npm run dev
```

## Frontend
```bash
- cd frontend
- npm install
- npm start
```
